description: Understanding how to work with any project.
---
If a project contains any `.eyg.json` files these can be imported and directly evaluated.
Use these scripts as a foundation when writing EYG scripts to work in such projects.

If a project contains a `gleam.toml` it is a Gleam project.
A tool to run tests in these project is currently unavailable but should be written soon.
