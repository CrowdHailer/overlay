# Overlay LLM

A unified API for LLM chat completion across providers.

Included providers are:
- [Mistral](https://mistral.ai/)
- [Ollama](https://ollama.com/)

## Development

```sh
gleam run   # Run the project
gleam test  # Run the tests
```
