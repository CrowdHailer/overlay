# Overlay Bun

A Cli implementation of the Overlay assistant.

## Installation

To build and install from source

```sh
./bin/compile
```

Copy to a directory on your $PATH
```sh
cp ./dist/overlay /usr/local/bin/overlay
```

## Development

```sh
gleam run . --read ../sibling --history
```

To run the tests.

```sh
gleam test  # Run the tests
```
